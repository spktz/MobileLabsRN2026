# Лабораторна робота №1
Цей мобільний застосунок є результатом виконання лабораторної роботи, спрямованої на опанування платформи Expo та вивчення базових принципів розробки на React Native. Основна мета проєкту полягала у створенні найпростішого додатка, ознайомленні з його файловою структурою та отриманні практичних навичок роботи з основними компонентами інтерфейсу.
У процесі розробки було реалізовано кілька функціональних екранів, серед яких домашня сторінка, профіль користувача та галерея зображень.

Для запуску проєкту необхідно завантажити репозиторій, перейти у відповідну директорію та виконати встановлення всіх пакетів через стандартний менеджер залежностей. Після ініціалізації сервера розробки за допомогою вбудованих інструментів Expo застосунок стає доступним для перегляду на реальних гаджетах або віртуальних пристроях.
У межах виконання роботи було розглянуто кілька основних підходів до запуску та тестування мобільних додатків. Використання реального фізичного пристрою разом із застосунком Expo Go дозволило оцінити реальну продуктивність програми та коректність відображення елементів на сенсорному екрані. Емулятор Android Studio було використано для перевірки адаптивності дизайну під різні параметри дисплеїв та системні налаштування без потреби у наявності декількох реальних смартфонів.

Завершальним етапом роботи стало опанування системи стилізації компонентів та налаштування навігації. Для створення зручного переходу між розділами було використано навігатор у вигляді верхніх вкладок, що забезпечило логічну структуру додатка. 
Отриманий досвід дозволив зрозуміти принципи побудови сучасних мобільних інтерфейсів та підготував базу для подальшого вивчення складніших архітектурних рішень у React Native.
