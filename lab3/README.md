### Лабораторна робота №3
Це мобільний додаток у форматі клікера, розроблений на React Native з використанням інструментів Expo. 
Головна мета роботи полягала у вивченні кастомних жестів та стилізації інтерфейсу, тому в проекті реалізовано навігацію через нижні вкладки для зручного переходу між ігровим екраном, списком завдань та налаштуваннями.

Для запуску проєкту потрібно завантажити всі файли, відкрити термінал і виконати команду npm install, щоб завантажити необхідні бібліотеки для роботи з жестами та навігацією. Після цього локальний сервер запускається командою npx expo start. 
Для перевірки роботи на реальному смартфоні використовується додаток Expo Go, де потрібно просто відсканувати QR-код, що з’явиться в терміналі.

Під час виконання лабораторної роботи було успішно реалізовано всі заплановані типи взаємодій, зокрема одинарний тап для нарахування очок та подвійний клік, для якого було налаштовано спеціальне очікування, щоб система їх не плутала. 
Також додано можливість утримувати об'єкт протягом трьох секунд для отримання бонусу, вільно перетягувати елемент по екрану, свайпати його в різні боки та масштабувати двома пальцями. 
Окремо було реалізовано сторінку завдань, де відстежується прогрес виконання всіх жестів, та сторінку налаштувань для перемикання між світлою та темною темами.

Вигляд додатка перевірявся кількома способами, включаючи віртуальний емулятор Android Studio, що зручно для написання коду, але сильно навантажує комп'ютер. 
Найбільш ефективним методом виявився запуск через Expo Go на смартфоні, оскільки це дозволяє реально відчути роботу жестів, особливо масштабування та перетягування. 
Хоча запуск через Wi-Fi іноді давав затримку, підключення через USB-кабель забезпечило найбільш стабільну роботу та швидку реакцію об'єктів на дотики.
