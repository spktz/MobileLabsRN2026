# Лабораторна робота №4
Цей мобільний додаток є повноцінним файловим менеджером, розробленим на React Native за допомогою Expo. У додатку реалізовано можливість повного керування локальними файлами та папками в пам'яті пристрою.

Для запуску проєкту необхідно завантажити репозиторій та виконати команду npm install для встановлення всіх залежних бібліотек, таких як система роботи з файлами та контекст безпечних зон екрана. Після завершення встановлення локальний сервер запускається командою npx expo start. Щоб відкрити додаток на віртуальному емуляторі Android Studio, достатньо натиснути клавішу "a" в терміналі, а для тестування на реальному смартфоні потрібно відсканувати QR-код через встановлений додаток Expo Go.

Під час виконання роботи було реалізовано та протестовано функціонал створення нових папок і текстових файлів, а також можливість їхнього видалення з попереднім підтвердженням. Додаток дозволяє переглядати детальну інформацію про кожен об'єкт: його назву, розмір у байтах, розширення та точний час останньої зміни. Також було додано функцію редагування текстових файлів через модальне вікно та відображення актуальної статистики пам'яті пристрою, включаючи загальний, зайнятий та вільний обсяги в гігабайтах. Весь інтерфейс додатка виконано в мінімалістичному темному стилі для зручності розробника.

Найбільш стабільним способом тестування файлової системи виявився запуск через USB-кабель, оскільки це забезпечує найшвидший запис та зчитування даних без затримок. Використання емулятора зручне для роботи з кодом на одному моніторі, але потребує багато ресурсів системи, тоді як Expo Go через Wi-Fi є найбільш мобільним варіантом для швидкої перевірки роботи функцій.
